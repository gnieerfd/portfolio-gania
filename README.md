# Gania Rafidah Portfolio

Static portfolio website built with plain HTML, CSS, and JavaScript.

## Structure

```text
.
├── index.html
├── assets/
│   ├── css/
│   │   └── style.css
│   ├── js/
│   │   └── script.js
│   └── images/
│       ├── logos/
│       ├── organizations/
│       ├── previews/
│       ├── profile/
│       └── projects/
└── docs/
```

## Deploy

For Vercel, use:

```text
Framework Preset: Other
Build Command: leave empty
Output Directory: .
```
